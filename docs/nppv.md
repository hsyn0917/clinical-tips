# 在宅 NPPV：適応と設定

## 拘束性胸郭疾患（RTD）

### 適応基準

!!! info "適応基準"
    **A. 自他覚症状あり** ＋ 以下のいずれか

    - **a.** 昼間覚醒時低換気：PaCO₂ **≧ 45 mmHg**
    - **b.** 夜間睡眠時低換気：SpO₂ < 90% が **5分以上継続**、または**全体の10%以上**を占める

    **B. 自他覚症状がなくても**、著しい昼間覚醒時低換気：PaCO₂ **≧ 60 mmHg**

    **C.** 高二酸化炭素血症を伴う**急性増悪入院を繰り返す**

### 生理的背景

- REM睡眠期は換気量が約 **40%** 低下する
- RTDはもともと一回換気量（TV）が少ないため、VTのわずかな低下でも肺胞換気量が著明に減少する
- → COPDと比べて昼間覚醒時のPaO₂から予測できないほど夜間に低酸素状態となりやすく、大半が昼間覚醒時高炭酸ガス血症を伴う

### 設定

!!! tip "モード選択"
    導入時に **T モード** と **ST モード** の両方を試す。患者が **T モードを好む場合は T を選択**。
    
    - 10年継続率：T モード（C,T）**41%** vs. ST モード（AC,ST）13%（呼吸筋休息の観点から T が優れている可能性）

**肺結核後遺症における代表的な設定（表2）**

| 項目 | 方法1（T モード） | 方法2（ST モード） |
|------|:---:|:---:|
| **モード** | T | ST |
| **IPAP** | 16 cmH₂O | 14 cmH₂O |
| **EPAP** | 2〜4 cmH₂O | 2〜4 cmH₂O |
| **換気回数** | 24 回/分 | バックアップ 20 回/分 |
| **吸気時間率（I%）** | 40% | バックアップ 40% |
| **最大吸気時間** | — | 1.2 秒 |
| **最小吸気時間** | — | 0.8 秒 |
| **吸入酸素量** | 酸素ポートより 1 L/分 | 酸素ポートより 1 L/分 |
| **（鼻口マスク時）** | 2 L/分 | 2 L/分 |
| **マスク** | 鼻マスクが主 | 鼻マスクが主 |
| | 開口著しい場合のみ鼻口マスク | 開口著しい場合のみ鼻口マスク |

!!! note "その他の調整ポイント"
    - **吸気トリガー感度**：auto trigger を生じない程度に鋭敏に
    - **呼気トリガー感度**：やや鈍く（吸気流速が早期に減弱しやすいため）
    - **Rise time**：0.1〜0.3 秒（やや長め）
    - **T モードで IPAP が低すぎると同調不良**になるため ST モードより 2〜4 cmH₂O 高めに設定

## COPD

### 導入基準

!!! info "導入基準"
    **自他覚症状あり** かつ 以下のいずれかを満たす場合

    1. PaCO₂ **≧ 55 mmHg**（室内気または処方量酸素下）
    2. PaCO₂ < 55 であるが、夜間の低換気による低酸素血症あり：SpO₂ < 90% が **5分以上継続**または**全体の10%以上**
    3. 安定期 PaCO₂ < 55 であるが、**高二酸化炭素血症を伴う増悪入院を繰り返す**

### 生理的背景・エビデンス

- 呼吸筋の負担軽減、呼吸調節系のリセットが示唆されている
- RCT のメタアナリシスでは、少なくとも3ヶ月の治療期間では PaCO₂・PaO₂・6MWD・QOL・PFT・呼吸筋力・睡眠効率のいずれにおいても有効性は認められなかった
- ただし一部には生存率を改善するとの報告もあり、現時点では安定期 COPD への長期 NPPV は一律に有効と言えない

### 設定

| 項目 | 設定 |
|------|------|
| **モード** | **S/T**（第一選択）。トリガーエラーが生じる場合は **Timed** へ変更を考慮 |
| **IPAP** | **高め（≧ 18 cmH₂O）**が重要。CO₂ 改善・死亡/入院率の改善と関連 |
| **使用時間** | 夜間睡眠時が望ましい。困難な場合は日中 **4時間以上** |
| **フォローアップ** | 導入後 **3〜4 ヶ月**で血液ガス・症状を評価し継続を判断 |

!!! note "高 IPAP に関するメタアナリシス"
    高 IPAP 群（≧ 18 cmH₂O）、アドヒアランス良好群（1日 5時間以上）、ベースライン CO₂ 高値群（≧ 55 mmHg）において、対照群と比べて有意な CO₂ 低下が示された。
    ただし、日本人は痩せ型が多く、高 IPAP の忍容性・有効性についてはさらなる検討が必要。

## 参考

??? info "急性期 COPD 増悪に対する NPPV の適応"
    以下のいずれか **1項目以上**を満たす場合

    - 呼吸性アシドーシス：pH **≦ 7.35** および/または PaCO₂ **≧ 45 mmHg**
    - 呼吸補助筋の使用・腹部奇異動作・肋間筋の陥没など、呼吸筋疲労または呼吸仕事量増加を示す臨床徴候を伴う **重度の呼吸困難**

??? info "在宅酸素療法（HOT）の適応"
    - PaO₂ **≦ 55 mmHg**
    - PaO₂ **≦ 60 mmHg** で、睡眠時または運動負荷時に著しい低酸素血症をきたす場合

    ※ 適応判定にはパルスオキシメータによる SpO₂ から PaO₂ を推測することも差し支えない
