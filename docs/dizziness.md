# めまい（ER初期対応）

## まず除外する危険徴候
- 局在神経症状（構音障害、複視、片麻痺、感覚障害）
- 激しい頭痛・頸部痛
- 歩行不能レベルの体幹失調
- 持続する嘔吐/脱水、意識障害

## 症候群で分ける（TiTrATE/GRACE-3の考え方）
1. **AVS**: 急性発症で持続（>24h）
2. **s-EVS**: 自然発作反復（診察時無症状あり）
3. **t-EVS**: 頭位変換などで誘発される短時間発作（BPPV）

## 30秒トリアージ
- 神経症状あり → 脳卒中動線
- 歩行不能/新規強い頭痛頸痛 → 中枢性を優先除外

## 診察の要点
- バイタル、起立性血圧
- 神経診察（眼球運動、小脳徴候、歩行）
- 眼振の性状
- AVSでは（習熟者なら）HINTS + 聴力
- t-EVSではDix-Hallpike（必要に応じRoll test）

## 画像の考え方
- CT単独で後方循環梗塞を除外しない
- 中枢疑いが残るならMRIや再評価を組み合わせる

## 治療の初期方針
- BPPV: Epley法など頭位治療
- 末梢性疑い: 制吐・補液、前庭抑制薬は短期（1–3日）
- 早期離床/前庭リハを意識

## 帰宅時のレッドフラッグ説明
- ろれつ不良、複視、麻痺/しびれ
- 強い頭痛・頸部痛
- 歩行不能、嘔吐持続、症状増悪
