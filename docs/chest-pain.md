# 胸痛（ER初期対応）

## 最初の目的
**見逃してはいけない致死的疾患を先に除外する。**

## 致死的胸痛 5本柱
1. 急性冠症候群（ACS）
2. 大動脈解離
3. 肺塞栓
4. 緊張性気胸
5. 食道破裂（Boerhaave）

## 初期対応（到着直後）
- ABC評価、バイタル、SpO2
- 12誘導ECGを速やかに（可能なら10分以内）
- モニター装着、静脈路
- 必要時採血（トロポニン、血算、生化、D-dimerは適応選択）
- 胸部X線（鑑別に応じて）

## 問診の軸（OPQRST + リスク）
- 発症様式（突然/労作時/安静時）
- 痛み性状（圧迫、裂ける、呼吸性、体位依存）
- 放散（背部、顎、上肢）
- 随伴症状（冷汗、呼吸困難、失神、神経症状）
- 既往/危険因子（CAD、喫煙、HT、DM、脂質異常、血栓リスク）

## 疾患別の示唆所見
- ACS: 圧迫感、冷汗、悪心、労作関連
- 解離: 突然最大強度、背部痛、血圧左右差、神経症状
- PE: 呼吸困難、胸膜痛、頻脈、血栓リスク
- 気胸: 片側胸痛+呼吸苦、呼吸音左右差

## 検査戦略（実務）
- ACS疑い: 連続ECG + 高感度トロポニン（0/1h or 0/2hプロトコル）
- 解離疑い高: 早期造影CT（必要時エコー併用）
- PE疑い: Wells/PERCで前確率評価→必要時造影CT

## すぐ介入が必要な場面
- STEMI: 速やかに再灌流動線
- 緊張性気胸: 即時脱気
- ショック/低酸素: 蘇生優先

## 帰宅可否の原則
- 致死的疾患の除外が十分
- バイタル安定
- 痛み増悪なく、再診導線が明確

## 帰宅時説明
- 胸痛再燃・増悪、呼吸困難、失神、冷汗を伴う痛みは即再受診
