# 間質性肺炎合併肺癌における細胞障害性抗癌剤の選択

出典：『間質性肺炎合併肺癌に関するステートメント2025（改訂第2版）』日本呼吸器学会

---

## 基本方針

- **推奨**：IPFを含む間質性肺炎合併肺癌患者に対して細胞傷害性抗癌薬を投与することを**提案**（推奨の強さ2、エビデンスの強さC）
- 急性増悪リスクと生命予後延長のバランスを評価し、十分なインフォームドコンセントのうえで実施
- BSCと比較し化学療法は予後を改善するが、急性増悪リスクは上昇する（Miyamoto 2023）

---

## 添付文書上の使用制限（重要）

| 区分 | 薬剤 |
|------|------|
| **禁忌**（間質性肺炎/肺線維症患者） | CPT-11 |
| **禁忌**（明らかで臨床症状のある IP/肺線維症） | GEM, AMR |
| **慎重投与** | CDDP/CBDCA、taxane（DTX・PTX・nab-PTX）、PEM 等 |

---

## 非小細胞肺癌（NSCLC）

### 一次治療：推奨レジメン

| レジメン | 位置づけ | 代表データ（ORR / PFS / OS / 急性増悪） |
|----------|----------|----------------------------------------|
| **CBDCA + nab-PTX** | **第一選択候補**（前向き第Ⅱ相の報告最多） | ORR 51%、PFS 6.2M、OS 15.4M、AE 4.3%（Kenmotsu 2019, n=92） |
| **CBDCA + weekly PTX** | 選択肢 | ORR 61〜69%、OS 10.6〜19.8M、AE 5.6〜12.1% |
| **CBDCA + S-1** | 選択肢（扁平・非扁平いずれも） | ORR 33%、OS 9.7〜12.8M、AE 6.1〜9.5% |
| **CBDCA + weekly PTX + Bev** | 非扁平上皮癌で選択肢 | ORR 52.9%、OS 12.9M、AE 5.9%（Omori 2023） |
| **CBDCA + nab-PTX + ニンテダニブ** | 非扁平上皮・GAP Stage Ⅰで選択肢 | J-SONIC試験：主要評価項目未達だが非扁平上皮でOS改善傾向（16.1 vs 13.1M） |

### 二次治療

- 移行率は約50〜60%。標準治療は未確立
- 選択肢：ドセタキセル、（nab-）パクリタキセル、S-1、ペメトレキセド、ビノレルビン
- ORR 8.6〜12.5%、OS 5.1〜8.8M、急性増悪率 14〜29%

## 小細胞肺癌（SCLC）

### 一次治療

| レジメン | 位置づけ | データ |
|----------|----------|--------|
| **CBDCA + エトポシド（CE療法）** | **第一選択** | ORR 88%、PFS 5.3M、OS 10.6M、AE 5.9%（Minegishi 2011 前向き）/ 松本2024：AE 0%、ORR 83.9% |
| **CDDP + エトポシド（PE療法）** | 選択肢 | AE 約11% |
| **CE + ニンテダニブ** | IPF合併例で選択肢 | NEXT-SHIP試験：AE 3.0%、ORR 68.8%（Ikeda 2024） |
| **CE + デュルバルマブ** | 軽症IP合併例で選択肢 | 重度AE未発生率 95.2%、PFS 5.5M（Shibaki 2024） |
| CDDP + イリノテカン（PI療法） | **避ける**（CPT-11 禁忌） | — |

### 二次治療

- **Sensitive relapse**：CE/PE 療法のリチャレンジが比較的安全（AE 0〜9%）
- **Refractory relapse**：パクリタキセルが選択肢（ただし AE 29% の報告あり）
- AMR・TOP は急性増悪リスクが高く（20〜33%）、原則回避または AMR は禁忌扱い

---

## 急性増悪リスク因子（治療前評価）

- HRCT での UIP パターン、線維化範囲 ≧25%
- %FVC 低値、%DLco ≦50%
- 高齢、ステロイド使用歴、PS 不良
- 血清 CRP 高値、Glasgow Prognostic Score ≧2
- サルコペニア
- リスクスコア：**ILD-NSCLC-GAP index**、Anti-cancer agent score 等

---

## 実地臨床のポイント

1. **ドライバー遺伝子変異陽性でも**、IP 合併例では EGFR-TKI 等は原則回避されるため、細胞障害性抗癌剤が中心となる
2. **一次治療の中心は CBDCA +（nab-）PTX ± Bev**、SCLC では **CE 療法**
3. 抗線維化薬（ニンテダニブ・ピルフェニドン）併用は、IPF 急性増悪の明確な予防効果は証明されていないが、非扁平上皮癌では予後改善傾向あり
4. 免疫チェックポイント阻害薬との併用は、軽症 IP 合併 SCLC で前向きデータあり（DREAM/Shibaki）
5. 治療前に患者・家族と予後と急性増悪リスクを十分に共有する
