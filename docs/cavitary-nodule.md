# 空洞性小結節の鑑別

## 鑑別リスト

| 疾患 | 所見の特徴 | 想起するポイント |
|------|-----------|----------------|
| **非結核性抗酸菌症（NTM）** | 右中葉・舌区中心、多発薄壁空洞、気管支拡張を伴う | 中高年女性、細長い体型、慢性経過 |
| **活動性肺結核**（気管支内播種含む） | 上葉優位、衛星病変、木の芽サイン（気管支内播種） | 発熱・盗汗・体重減少、接触歴 |
| **細菌性気道感染 / 慢性化膿性気道疾患** | 厚壁空洞、周囲浸潤影、air-fluid level | 急性経過、膿性痰、既存肺疾患（気管支拡張症など） |
| **敗血症性肺塞栓** | 両肺末梢側・多発・楔形、feeding vessel sign | 静脈内薬物使用、中心静脈カテーテル、三尖弁心内膜炎 |
| **真菌感染**（アスペルギルス・クリプトコッカスなど） | halo sign（侵襲性アスペルギルス）、air crescent sign | 免疫不全（好中球減少、ステロイド、HIV） |
| **GPA（多発血管炎性肉芽腫症）など血管炎** | 両肺多発・厚壁・辺縁不整空洞、出血を伴うことあり | 副鼻腔炎・血尿の合併、PR3-ANCA陽性 |
| **空洞性肺転移** | 多発・両肺・大小不同、壁厚さまざま | 悪性腫瘍の既往（特に扁平上皮癌、肉腫） |

---

## 補足

### 壁の厚さによる見当

| 壁厚 | 傾向 |
|------|------|
| 薄壁（≤4 mm） | NTM、嚢胞性転移（まれ） |
| 中間（5〜15 mm） | 結核、真菌、炎症 |
| 厚壁（>15 mm） | 化膿性感染、癌、GPA |

### 追加で確認すること

- **分布**：上葉優位 → 結核・NTM、下葉末梢 → 敗血症性塞栓、胸膜下 → GPA
- **経過**：急性（数日） → 細菌・敗血症性塞栓、亜急性（週〜月） → 真菌・結核・NTM・悪性
- **免疫状態**：免疫正常 vs 免疫抑制（好中球減少・ステロイド・HIV）
- **随伴症状**：副鼻腔炎・腎炎 → 血管炎、注射痕・弁膜症 → 敗血症性塞栓
