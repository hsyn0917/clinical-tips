## 特に注意が必要な薬剤のまとめ

### 腎機能低下時に減量・禁忌となる薬剤

| 薬剤名 | 注意点 |
|--------|--------|
| **シスプラチン** | CCr≦30で原則投与不可。最も腎毒性が高い白金製剤 |
| **カルボプラチン** | カルバートの式による個別用量算出が必須。GFR値を正確に評価する |
| **ネダプラチン** | 重篤な腎障害には禁忌 |
| **TS-1** | CCr<30で禁忌。ギメラシルの腎排泄低下により5-FU濃度が上昇し重篤な骨髄抑制のリスク |
| **ペメトレキセド** | CCr<45ではデータ不十分のため使用を推奨しない |
| **エトポシド** | 腎機能低下時は75%に減量 |
| **イホスファミド** | 腎・膀胱に重篤な障害がある場合は禁忌 |

### 腎機能低下時も減量不要な薬剤

| 薬剤名 | 備考 |
|--------|------|
| ドセタキセル | 肝代謝が主のため腎機能の影響を受けにくい |
| パクリタキセル | 同上 |
| ビノレルビン | 同上 |
| ゲフィチニブ | 腎排泄の寄与が少ない |
| エルロチニブ | 同上 |
| クリゾチニブ | 同上 |
| ベバシズマブ | 抗体製剤のため腎排泄の影響を受けにくい |
