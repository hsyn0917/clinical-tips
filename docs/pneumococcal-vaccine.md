# 肺炎球菌ワクチン接種方針（2026年4月〜）

!!! info "2026年4月 定期接種ワクチン変更"
    高齢者の定期接種が **ニューモバックス® NP（PPSV23）→ プレベナー20®（PCV20）** に切り替わった。

---

## ワクチン一覧

| 略称 | 一般名 | 商品名 | 製造販売 | 区分 |
|------|--------|--------|---------|------|
| **PCV20** | 沈降20価肺炎球菌結合型ワクチン | **プレベナー20® 水性懸濁注** | ファイザー | **定期接種（2026年4月〜）** |
| **PCV21** | 沈降21価肺炎球菌結合型ワクチン | **キャップバックス® 筋注シリンジ** | MSD | 任意接種（2025年10月〜） |
| PPSV23 | 23価肺炎球菌莢膜ポリサッカライドワクチン | **ニューモバックス® NP** | MSD | 任意接種（定期接種から除外） |
| PCV15 | 沈降15価肺炎球菌結合型ワクチン | **バクニュバンス® 水性懸濁注** | MSD | 任意接種（使用機会は減少） |

---

## 定期接種（2026年4月〜）

### 対象者
- **65歳**の方（基本）
- **60〜64歳**で以下のいずれかに該当する方
    - 心臓・腎臓・呼吸器機能に著しい障害（日常生活がほとんど不可能な程度）
    - HIVによる重度免疫不全

### 接種方法
- ワクチン：**プレベナー20®（PCV20）**
- 用量・経路：**0.5 mL、筋肉内注射 1回**（皮下注は不可）
- 費用：公費（自治体負担）

!!! warning "接種経路の変更点"
    旧・ニューモバックス® NP（PPSV23）は筋注・皮下注どちらも可だったが、プレベナー20®（PCV20）は **筋肉内注射のみ**。

---

## 任意接種の選択肢

### PCV21（キャップバックス®）— 成人向けに特化した新ワクチン

- 2025年8月承認、同年10月より接種開始
- **成人の市中肺炎原因血清型の71.9%、IPDの80.3%** をカバー（国内データ）
- 高齢者で特に問題となる血清型を重点的にカバーする設計
- **1回接種**で完結（追加・再接種不要）

### PCV20（プレベナー20®）の任意接種利用

- 定期接種の対象外（66歳以上、または未接種の65歳以外）の方には任意接種として選択可能

---

## 接種歴に応じた推奨フロー

### ① 未接種の方（65歳時）
```
→ プレベナー20®（PCV20）を定期接種（公費）
  以後の追加接種は原則不要
```

### ② ニューモバックス® NP（PPSV23）接種済みの方
```
PPSV23接種から1年以上経過後
  → PCV20（プレベナー20®）または PCV21（キャップバックス®）を任意接種
  → 以後のニューモバックスの5年ごと再接種は不要
```

!!! note "結合型ワクチン接種後の再接種"
    PCV20またはPCV21を1回接種した後は、原則として追加・再接種は不要とされている（2025年改訂ガイドライン）。

### ③ 結合型PCV（プレベナー20®またはキャップバックス®）接種済みの方
```
→ 追加接種は原則不要
```

## PCV20 vs PCV21 の比較

| 項目 | PCV20（プレベナー20®） | PCV21（キャップバックス®） |
|------|----------------------|--------------------------|
| 価数 | 20価 | 21価 |
| 設計 | 小児〜成人兼用 | **成人・高齢者特化** |
| 定期接種 | **65歳 公費対象** | 対象外（任意接種のみ） |
| 接種経路 | 筋肉内注射 | 筋肉内注射 |
| 高齢者IPDカバー率 | — | 80.3%（国内データ） |
| 費用 | 定期接種：公費、任意：自費 | 自費のみ |

---

## 費用対効果

厚生科学審議会資料では「65歳にPCV20を接種する戦略は、効果が向上しかつ費用が削減される（Dominant）」とされ、現行の定期接種プログラムの中で最も費用対効果に優れる戦略と評価されている。
