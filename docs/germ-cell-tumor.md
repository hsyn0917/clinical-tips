# 精巣腫瘍・胚細胞腫瘍

## 概要

- **15〜40歳に好発**、この年齢層の固形癌で最多
- 有病率は1〜2人/10万人（年間）
- 無痛性陰嚢腫瘤が典型（>90%）
- 手術＋化学療法＋放射線で高確率に完治が見込める

---

## 臨床像

### 典型症状

- 無痛性陰嚢腫瘤（>90%）
- 陰嚢の重感・違和感
- 転移に伴う症状：背部痛（後腹膜リンパ節）、呼吸症状（肺転移）、頸部腫瘤（縦隔転移）

### 鑑別疾患

| 疾患 | 鑑別点 |
|------|--------|
| 精巣上体炎 | 発熱・疼痛あり、Prehn徴候陽性、若年男性で性感染症リスク |
| 陰嚢水腫 | 透光性（transillumination）陽性、精巣自体は正常 |
| 精索静脈瘤 | 「虫の袋」状、立位で増悪、Valsalvaで拡張 |
| 鼠径ヘルニア | 腸管音、体位変換で変化 |

### 転移部位

- **後腹膜リンパ節**（最初の転移部位）→ 縦隔・左鎖骨上リンパ節
- **肺**（血行性）
- 肝・骨・脳（頻度低いが予後不良）

---

## 組織分類

```
精巣腫瘍
├── 胚細胞腫（90〜95%）
│   ├── 精上皮腫 seminoma（約半数）
│   └── 非精上皮腫 non-seminoma
│       ├── 胎児性癌（embryonal carcinoma）
│       ├── 卵黄嚢腫瘍（yolk sac tumor）
│       ├── 奇形腫（teratoma）
│       ├── 絨毛癌（choriocarcinoma）
│       └── 複合組織型（mixed GCT）
├── 精索/性腺間質腫瘍
│   ├── ライディヒ細胞腫（アンドロゲン産生）
│   ├── セルトリ細胞腫
│   └── 顆粒膜細胞腫（エストロゲン産生）
└── その他：悪性リンパ腫、白血病、転移性腫瘍
```

## 診断

### 腫瘍マーカー

| | 精上皮腫 | 胎児性癌 | 奇形腫 | 卵黄嚢腫瘍 | 絨毛上皮腫 |
|--|:---:|:---:|:---:|:---:|:---:|
| **hCG** | ± | ++ | + | + | +++ |
| **AFP** | **−** | ++ | + | +++ | − |

!!! warning "重要"
    **AFPが上昇した場合、精上皮腫の診断は否定される**。non-seminomaを含む混合型を疑う。

- LDHも採血する（IGCCC予後分類に必要）
- マーカー正常でも精巣腫瘍を否定できない（精上皮腫の多くはAFP陰性）

### HCG測定試薬の選択

IGCCC予後分類には **トータルHCG（mIU/mL単位）** を使用する。

| 種別 | インタクトHCG（αβ） | トータルHCG（αβ＋β） | 遊離β-HCG |
|------|:---:|:---:|:---:|
| 測定場所 | 院内（当日） | 院内（当日） | 外注（2〜4日） |
| 単位 | mIU/mL | mIU/mL | ng/mL |
| IGCCC分類への使用 | ○ | ○ | **× 使用不可** |
| 交差反応（LH/FSH/TSH） | あり | あり | 少ない |

!!! info "遊離β-HCGはng/mLのため、IGCCC分類のmIU/mL基準値と単位が異なり使用できない"

### 画像所見

| | 精上皮腫 | 非精上皮腫 |
|--|:---:|:---:|
| **エコー** | 均一低エコー | 不均一 |
| **MRI（PDWI/T2WI）** | 均一・低信号 | 不均一・高信号 |
| **血流・造影効果** | 軽度 | 不均一 |

- **精巣エコーが第一選択**（感度・特異度ともに高い）
- CT（胸腹骨盤）で転移検索（後腹膜リンパ節・肺・肝・骨）

## 治療

### 高位精巣摘除術（High inguinal orchiectomy）

- **経鼠径アプローチ**で精索を内鼠径輪レベルで結紮し精巣を摘除
- 病理組織型を確認して以後の方針を決定

!!! danger "禁忌"
    **陰嚢切開・経陰嚢生検は禁忌**。陰嚢・鼠径リンパ節への転移リスクが上昇し、治療が複雑化する。

### 術後治療（組織型・ステージ別）

| 組織型 | ステージI（限局） | 進行例 |
|--------|-----------------|--------|
| **精上皮腫** | 経過観察 / 放射線療法 / カルボプラチン1コース | **BEP療法** |
| **非精上皮腫** | 経過観察 / 後腹膜リンパ節郭清（RPLND） | **BEP療法** |

**BEP療法**: ブレオマイシン（B）＋ エトポシド（E）＋ シスプラチン（P）

## IGCCC予後分類（初回化学療法前）

### Good prognosis

| | 非セミノーマ | セミノーマ |
|--|-------------|----------|
| **条件** | 精巣または後腹膜原発、肺以外の臓器転移なし、AFP＜1000 ng/mL、hCG＜5000 IU/L、LDH＜1.5×ULN | 原発巣問わず、肺以外の臓器転移なし、AFP正常（hCG・LDHは問わない） |
| **5年非再発率** | 89% | 82% |
| **5年生存率** | 92% | 86% |

### Intermediate prognosis

| | 非セミノーマ | セミノーマ |
|--|-------------|----------|
| **条件** | 精巣または後腹膜原発、肺以外の臓器転移なし、AFP 1000〜10000 ng/mL または hCG 5000〜50000 IU/L または LDH 1.5〜10×ULN | 原発巣問わず、肺以外の臓器転移あり、AFP正常 |
| **5年非再発率** | 75% | 67% |
| **5年生存率** | 80% | 72% |

### Poor prognosis

| | 非セミノーマ | セミノーマ |
|--|-------------|----------|
| **条件** | 縦隔原発、または肺以外の臓器転移あり、またはAFP＞10000 ng/mL、hCG＞50000 IU/L、LDH＞10×ULN | **該当なし** |
| **5年非再発率** | 41% | — |
| **5年生存率** | 48%（近年は70〜80%に改善） | — |

!!! info "近年の成績"
    上記数値は1997年報告時のもの。特に非セミノーマのpoor prognosis群は近年の治療改善により、全生存率が約70〜80%まで改善している。

## Burned-out Tumor（焼け落ち型精巣腫瘍）

- **原発巣が自然退縮**し、組織学的に認識困難となった状態
- 転移巣（後腹膜リンパ節・縦隔・肺）のみが発見されることがある
- 精巣内には**線維化・石灰化・瘢痕（scar）のみ残存**
- 腫瘍マーカー（AFP/hCG）上昇を伴う場合は積極的に疑う

### 診断の手がかり

- **精巣エコー**：低エコー病変、石灰化、瘢痕（不均一）
- 原発巣が確認できなくても、転移巣の組織型（胚細胞腫瘍）が確認されれば診断
- 原発不明の後腹膜腫瘤・縦隔腫瘤では、若年男性なら必ずGCTを鑑別に入れる

### 治療方針

原発巣が自然退縮していても、**転移巣に対して化学療法（BEP）を施行**する。精巣自体も摘除（or 注意深い経過観察）。

## ERでの実践的アプローチ

### 精巣腫瘍を疑ったら

1. **精巣エコー**（第一選択）
2. **腫瘍マーカー採血**：AFP、hCG（インタクトorトータル）、LDH
3. **泌尿器科コンサルト**
4. 精巣上体炎との鑑別：腫瘍は通常無痛・無発熱
5. 転移症状の評価（背部痛・呼吸症状・頸部リンパ節）

!!! danger "禁忌"
    **陰嚢切開・生検は絶対に行わない**

### 精巣上体炎との鑑別

| | 精巣腫瘍 | 精巣上体炎 |
|--|:---:|:---:|
| 疼痛 | 通常なし | あり |
| 発熱 | なし | あり |
| Prehn徴候（挙上で軽快） | 陰性 | 陽性 |
| エコー | 精巣内低エコー病変 | 精巣上体腫大・充血 |
| 年齢 | 15〜40歳 | 性活動期（幅広い） |
